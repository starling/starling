Starling

Get Version

1.0.0

What

Starling is a light-weight server for distributed message passing.

Installing

sudo gem install starling

The basics

  1. Start the Starling server.
  2. Put messages onto a queue from a client whose work you’d like to offload to a deferred process.
  3. Run a daemon process to take messages off the queue, and process as time allows.

Demonstration of usage

  1. Start the Starling server:

    starling -h 192.168.1.1 -d
  1. Queue messages for later processing:

    require 'memcache'
    starling = MemCache.new('192.168.1.1:22122')
    starling.set('my_queue', 12345)
  1. Fetch messages from the queue and do something with them:

    require 'memcache'
    starling = MemCache.new('192.168.1.1:22122')
    loop { puts starling.get('my_queue') }
